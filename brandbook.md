## Values 04

##### We simplify

```
Yuh’s brand promise is simplicity and
effortlessness. We transform everything
related to money into simple actions and
intuitive movements. More broadly, our goal
is to create peace of mind and positivity
towards the future.
```
```
Thus, we are trying to avoid overly
technical terms and long sentences. Pick
light and easy words. Use an active and
conversational tone. Make it sound simple,
not formal. Stay short and concise.
```
```
Make the solution seem always at
hand and easy to grab! Apple brought
technology into daily life. We do the same
with banking!
```

## Values 08

##### But we don’t

```
We speak in a conversational tone,
but we don’t go for slang or teenage
language. First, because our users
are not all teenagers. Then, because
we take money seriously.
```
```
We don’t force creativity. Too many
pictos on the app complicate the app...
and too many metaphors and puns
complicate the voice! They must come
with a purpose.
```
```
We don’t talk to our users like clients.
The customer-company relation is too
hierarchical and from the past. We are
an app, an enabler: in that case, consider
our “clients” as users (Yuhsers).
```
```
We are not a bank! We are an app.
It’s not just a state of mind. It’s also
a legal obligation: we don’t have
the right to use the word “banking”.
However you could say financial app,
Yuh app, or simply app!
```

##### General use

### 14 rules to reach Nirvana (of writing)

4. Spacing

```
Currencies
Use a non-breaking space between
numbers and currencies: 20 CHF
```
```
Text use
No space before punctuation in any
language
```
5. URL

```
Without the use of 3w: yuh.com
```
6. Phone number

```
Phone number: +41 xx xxx xx xx
```
7. Currencies

```
Currencies are always placed after
numbers with a non-breaking space:
e.g. 20 CHF, 4.90 CHF
Exception: To come straight to the point
in videos, pictures or any other kind of bold
advertising we can write it like this: 0.-
```
1. Form – formal/informal

```
General use of the informal form of you,
except for legal documents.
DE: du
FR/IT: tu
```
2. Capital letters

```
General
Start your text with a capital letter in every
language (Headline, Subline, Bodycopy,
CTA, descriptive text, USP, etc.)
```
```
App’s pillars (Pay, Save & Invest)
If they are used as a pillar and not a verb,
write them with a capital letter. They can be
in non-capital letters when they are used in
a sentence
```
```
Proper nouns
Write with a capital letter
```
```
French
Accent is used on capital letters
```
3. Punctuation

```
Headlines and bodycopies
Prohibited: no use of final full stop
Accepted: comma / exclamation mark /
question mark
```
```
Descriptive text, bodycopy or USP Always
use punctuation
```
```
Quotation marks
English: “ ”
German: „ “
French/Italian: « »
```
```
Colon rules
English/German: Use a capital letter
if the first word after the colon is a
proper noun or introduces a grammatically
complete sentence. Continue
with a lower case letter if the colon
introduces an incomplete sentence.
French/Italian: Always continue with
a lower case letter, except if the first word
after the colon is a proper noun.
```
```
Put an apostrophe every 3 digits,
starting from a 4-digit number:
e.g. 1’000’
```
```
Decimals
Use a dot for all languages:
e.g. 4.90, 55.60, 0.
```
8. Numbering

```
Use superscript for numbers
English: 1st / 2nd
French: 1er / 1re, 2e, 3e
```
```
If impossible to use superscript:
English: 1st / 2nd
French: 1er / 1re, 2e, 3e
```
9. Wording

```
Prohibited words
Bank/ Banking app: never use it to speak
about Yuh (Yuh isn’t a bank).
Use alternatives such as financial
for all languages
```
```
Use Yuh/You puns sparingly
```

11. Inclusive writing

```
Wherever possible, use gender neutral ex-
pressions. Should this not be possible here
is how we suggest the use of an inclusive
writing:
```
```
English
use of / (ex. waiter/ress,
salesman/woman)
```
```
German
use of * (ex. Freunde*innen, die*der
Lehrer*in; In der Checkliste findet ein*e Stu-
dierende*r, was er*sie in den ersten Wochen
erledigen muss.)
```
```
French
use of -e (ex. l’étudiant-e, un-e étudiant-e,
utilisateur/trice, détenteur/trice); plural form:
ex. les ami-es, les client-es
```
```
Italian
use of / (ex. l’amico/a, un/a amico/a, profes-
sore/essa); plural form: ex. gli/le italiani/e,
molti/e oratori/trici
```
12. Language exclusivities

```
German
Hyphen
Without hyphen: Yuh Crew / Yuh Key (words
coming from English)
With hypen: Yuh-App / Yuh-Konto /
Yuh-Familie (German words)
```
```
SS
For the Swiss market we write SS
instead of β
Example: Der grösste Finanzmarkt
```
```
English
Yuh prefers the use of British English, hence
also the use of “s” instead of “z”
Example: legalise, normalise,
customise...
```
13. Use of the ampersand (&)

```
This sign should only be used to combine
two proper or company names (e.g. Villeroy
& Boch, Schmidt & Schmidt) and needs spa-
cing before and after. It should not be used
as a substitute for the word “and” (as well as
und, et, e in the other languages). However,
exceptions are possible due to lack of space.
```
